Football Season Simulation

About this project

The goal of this project is to simulate in a really simple way the entire football season including the playoffs.

What are my goals?

I want to focus on optimizing the way my code works, object-oriented programming practice, and just having as much fun as possible!

Project

  • Simulate football season of n teams.
  • Teams will play in a round-robin tournament system. Where 4 teams with most points will play in the play-off in a single-elimination (knockout) system.
    The total number of games would be G = (n*(n-1))/2 + N - 1 where N is the number of playoff teams.
  • For winning a game team will get 3 points, the tie would give 1 point, and there are no points for losing a game!
  • Each team must have:
    • Name (doesn't need to be unique, but should be).
    • Unique ID.
    • Amount of points gathered in this season.
