Football Season Simulation

About this project

The goal of this project is to simulate in a really simple way the entire football season.

What are my goals?

I want to focus on optimizing the way my code works, object-oriented programming practice, and just having as much fun as possible!

Project

  • Simulate football season of n teams where n >= 4.
  • Teams will play in a round-robin tournament system.
    The total number of games would be G = (n*(n-1))/2.
  • Each team must have:
    • Name (doesn't need to be unique, but should be).
    • Unique ID.
    • Skill Points - randomly generated.
    • Win Counter
    • Lose Counter
    • Draw Counter
    • League Points - Amount of points gathered in this season.
    • Scored Goals
  • Games - each team will have a chance of scoring every GameTurns attempts where
    GameTurns = rand(1,(Team1Skill(with more points) - Team1Skill(with less points))%10)
    or in case if teams skill points are equal GameTurns = rand(1,10). Game would look like:
    for GameTurns 
    temp1 = Team1Skill
    temp2 = Team2Skill
    while temp1>=0 and temp2>=0
        temp1 -= random from 1 to 5;
        temp2 -= random from 1 to 5;
    if temp1<=0 and temp2<=0
        no goals
    if temp1<temp2
        team1 scores
    else 
        team2 scores
  • Results of game:
    • For winning a game team will get 3 points, the tie would give 1 point, and there are no points for losing a game!
